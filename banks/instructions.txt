Mega Man 3 — Ghidra Import Instructions
========================================

ROM: Mega Man 3 (USA), Mapper 4 (MMC3)
PRG: 16 x 16KB banks (256KB), CHR: 16 x 8KB banks (128KB)
Vectors: NMI=$C000  RESET=$FE00  IRQ=$C143

Bank Layout (recompiler numbering, 16KB pairs of 8KB MMC3 banks):
  bank00.bin — 8KB banks $00+$01 (switchable, mapped at $8000-$BFFF)
  bank01.bin — 8KB banks $02+$03
  bank02.bin — 8KB banks $04+$05
  bank03.bin — 8KB banks $06+$07
  bank04.bin — 8KB banks $08+$09
  bank05.bin — 8KB banks $0A+$0B
  bank06.bin — 8KB banks $0C+$0D
  bank07.bin — 8KB banks $0E+$0F
  bank08.bin — 8KB banks $10+$11
  bank09.bin — 8KB banks $12+$13
  bank10.bin — 8KB banks $14+$15
  bank11.bin — 8KB banks $16+$17
  bank12.bin — 8KB banks $18+$19
  bank13.bin — 8KB banks $1A+$1B
  bank14.bin — 8KB banks $1C+$1D (main sprite/enemy AI)
  bank15.bin — 8KB banks $1E+$1F (FIXED bank, always at $C000-$FFFF)
  chr00-15.bin — CHR ROM banks (tile graphics, not code)

NOTE: MMC3 actually uses 8KB PRG banking. The 16KB banks here pair
two 8KB banks for Ghidra convenience. The first 8KB in each file maps
to $8000-$9FFF, the second 8KB to $A000-$BFFF (or $C000/$E000 for fixed).

=====================
STEP 1: Create Project
=====================
  File > New Project > Non-Shared Project > pick a folder > name it "MegaMan3"

=====================
STEP 2: Import Fixed Bank (bank15.bin) — DO THIS FIRST
=====================
  File > Import File > browse to bank15.bin
  In the import dialog:
    - Language: click "..." button, search "6502", pick "6502:LE:16:default"
    - Format: Raw Binary
    - Click "Options..." button at bottom
      - Set "Base Address" to: C000
    - Click OK, then OK again
  Double-click the imported file in the project window to open it in CodeBrowser.
  When asked "Analyze now?" click YES, accept defaults, click Analyze.

=====================
STEP 3: Import Switchable Banks (as needed)
=====================
  File > Import File > browse to bankNN.bin
  Same settings as above EXCEPT:
    - Base Address: 8000
  Each switchable bank maps to $8000-$BFFF.
  Import as separate programs in the same project.

  Priority banks to import for debugging current issues:
    bank14.bin — Main sprite/enemy AI, player routines ($8000-$BFFF)
    bank00.bin — Enemy attribute tables + code
    bank09.bin — Code (stage loading?)

=====================
STEP 4: Define Entry Points (all in bank15.bin / fixed bank)
=====================
  G > FE00 > Enter    then    F          (RESET vector)
  G > C000 > Enter    then    F          (NMI vector)
  G > C143 > Enter    then    F          (IRQ vector)

  Key utility functions in fixed bank:
    $FF6B — select_PRG_banks (bank switching routine)
    $FF41 — select_CHR_banks
    $FF3C — update_CHR_banks
    $FF90 — play_sounds
    $C8D0 — secondary init (post-reset, menu/game setup)
    $C545 — process_sprites
    $C59D — check_new_enemies
    $C5E9 — spawn_enemy

=====================
OTHER USEFUL HOTKEYS
=====================
  G         — Go to address
  F         — Create function at cursor
  D         — Disassemble bytes at cursor
  L         — Rename/label the current address
  ;         — Add a comment
  Space     — Toggle listing/decompiler view
  X         — Show cross-references

=====================
MM3 MEMORY MAP (key RAM addresses)
=====================
  $0022 — Current stage number
  $0031 — Game mode / state
  $0046 — Sub-mode
  $00A0 — Current weapon
  $00AE — Lives remaining
  $00FF — NMI enable flag (written to $2000)
  $00F4 — PRG bank select value (used by select_PRG_banks)
  $0300-$05FF — Object/entity arrays (position, type, state)
